== Apache KafkaRecipes ==

This project provides sample recipes for usage of Apache Kafka Message broker.

=== Requirements: ===
* Apache Maven version 3
* Apache Kafka 0.10.1.1
* Apache ZooKeeper 3.4.9
* JDK 8
